Advanced Example

In this example we showcase how you can use conditional validation using when to have rules that only apply when certain fields or values are provided. All validation logic is handled inside of basic-example.ts

Try checking the VIP checkbox and hitting submit. Then uncheck it. Notice the error conditionally displays?

Please also note the use of a custom element. Using newInstanceForScope will allow validation bindings to be used within child custom elements.
