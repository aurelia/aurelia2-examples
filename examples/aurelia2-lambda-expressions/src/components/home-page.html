Lambda Expressions

In this section we are going to learn how to use lambda expressions to call functions and perform other tasks from within our views. All code is defined inside of home-page.html

Repeat.for Lambda Expressions

Array.filter

In the following example we filter names by their starting letter "J". We use Array.filter followed by the startsWith function.

  • ${name}

Array.filter and Array.sort

We can combined multiple functions inside of our Lambda Expressions as well. In this example we filter out any names without a name value, we can use Array.sort to sort all names by age.

  • ${name.name}

Array.slice

In this example we take a portion of our names array using Array.slice. We'll take the first three names using items.slice(0,3)

  • ${name}

Interpolation Lambda Expressions

We can call lambda functions inside of interpolation in our templates from within \${}. The possibilities this opens up mean you can avoid the need to boilerplate and do it all in the template instead.

Array.join

We can use the Array.join method to create a comma separated list of our names from our names array.

${names.join(', ')}

Array.reduce

We can use the Array.reduce method to reduce an array of values down to a singular value from our products array.

Total: ${products.reduce((sum, product) => sum + product.price, 0)}
