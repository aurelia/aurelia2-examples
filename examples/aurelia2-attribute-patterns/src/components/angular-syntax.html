Angular Syntax

All bindings in this page are leveraging Angular banana in a box [()] syntax. To see how it pieces together, check out angular-syntax.html

AttributePattern logic is registered inside main.ts.

Text input

Typing into the following form input using [(value)] for binding should result in the value being dislpayed beneath.

Form value is: ${inputValue}

Checkbox

We can also bind to the checked value of a checkbox using [(checked)].

Form value is: ${inputChecked ? 'checked' : 'unchecked'}
