Basic Animation

In this section, we show you how to leverage the native animation API inside your Aurelia applications. All animation logic is defined inside of basic-animation.ts

Fade In Animation

A simple fade in animation that animates opacity of 0 to 1.

Fade Out Animation

A simple fade out animation that animates opacity of 1 to 0.

Infinite Rotate Animation

An endless rotation animation that animates the rotate property. Unlike the other animations, we can cancel this one.

Flip (vertical) Animation

A simple vertical flip animation that animates the rotateX value.

Flip (horizontal) Animation

A simple horizontal flip animation that animates the rotateY value.
