Global CSS Styles

In some instances you might be dealing with third-party libraries such as Bootstrap which rely on styles being present in the global scope. This can be challenging in a ShadowDOM environment which does not have global styles (everything is walled off).

Fortunately, Aurelia allows us to implement global styles where we register the style plugin inside of main.ts. You will notice there is a configuration property sharedStyles which is an array of imported styles. Every component gets these shared styles injected.

You might have noticed, but this example application uses Bootstrap for the styles (grid system and all). As you can see, it is possible for Bootstrap and Shadow DOM to coexist.
