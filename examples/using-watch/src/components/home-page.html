Using Watch

In this section we are going to learn how to use the powerful @watch decorator to watch data. All code is defined inside of home-page.ts

Watching class properties

You can watch a property in your view-model and run a function when it changes. The console on the right hand side will update as you type.

Observing array length changes

The @watch decorator also allows us to observe the length of an array. Click the button to add a new item to the items array.

Complex watch expression

We can use array methods to observe for changes to our array and values inside of it. Click the button to add a new item to the items2 array. Our watch expression will return the values in our array separated by a comma.

Rob Watcher

Type in the value "Rob" (without the quotes) to see a special surprise in the console to the right. Once you've typed it, look what happens when you delete the value.
